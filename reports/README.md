# MSc Data Analytics – [Data Design module]

## Overview
This repository contains academic coursework completed as part of the **MSc Data Analytics & IT Security Management** programme at **Arden University**.

The work focuses on applying data analytics techniques to real-world scenarios, including data preparation, analysis, interpretation, and communication of insights to support evidence-based decision-making.

---

## Module Information
- **Programme:** MSc Data Analytics & IT Security Management  
- **Module:** Data Design  
- **Institution:** Arden University  
- **Academic Year:** 2024–2025  
- **Assessment Type:** Report / Essay / Case Study / Portfolio  

---

## Objectives
The objectives of this module assignment were to:
- Apply structured data analytics methodologies to real-world datasets
- Perform data cleaning, exploration, and analysis
- Interpret analytical results using statistical reasoning
- Translate findings into clear business or policy insights
- Communicate results effectively to technical and non-technical stakeholders
